AntNews 是一款资讯门户风格 WordPress 主题，适合博客、资讯站、内容社区和轻量门户。

== 版本规则 ==
初始版本为 6.6.1。
每修改一次，小版本升级，例如 6.6.1 -> 6.6.2。
增加功能时，中版本升级，例如 6.6.x -> 6.7.0。
重构主题架构时，大版本升级，例如 6.x.x -> 7.0.0。

== 主要功能 ==
* 顶部通知条
* 粘性 Header 和主导航
* 首页轮播和右侧推荐卡片
* 专题分类推荐
* 双栏资讯流布局
* 文章列表、归档、搜索、详情、页面和 404 模板
* 自定义颜色、多条通告、轮播分类、HTML 广告和页脚版权
* AntNews 文章列表、广告图、快讯、热门 Tab、产品双列小组件
* 移动端侧滑导航和底部工具栏
* 分类专题封面设置

== 安装方式 ==
1. 在 WordPress 后台进入“外观 > 主题 > 添加新主题 > 上传主题”。
2. 上传 AntNews.zip。
3. 启用主题后进入“外观 > 自定义 > AntNews 主题设置”配置首页和视觉。
4. 在“外观 > 菜单”配置主导航和页脚导航。
5. 在“外观 > 小工具”配置主侧栏。
